# 4. 들여쓰기

1. 괄호를 사용하지 않고 들여쓰기로 구분
2. 제어문, 함수, 클래스 뒤에 `:` 표시 후 코드 작성
3. 들여쓰기 띄어쓰기 잘못하면 SyntaxError

```python
def example(word) : # {} 중괄호 쓰지 말기
    if word in 'abc' : # 조건에 () 괄호 쓰기 말기
        print("{0}가 존재합니다!".format(word))	# 끝에 ; 세미콜론 쓰지 말기, 들여쓰기 주의
```
